## What Is This?

This is a structured, interactive reference that maps out **234 Python and programming concepts** across 3 learning layers:

| Layer | Colour | What It Covers |
|-------|--------|----------------|
| **L1** — Core Programming | 🔴 Red | Language-agnostic fundamentals that apply to *any* programming language |
| **L2** — Python Ecosystem | 🟢 Teal | How Python works as a platform — environments, tools, packaging |
| **L3** — Python In Depth | 🟡 Amber | Python-specific syntax, features, and standard library in detail |

Concepts are grouped into **20 themed clusters** that relate ideas across layers — for example, the *Variables & Data* cluster connects the general concept of data types (L1) all the way through to Python's `typing` module and `enum` (L3).

---

## Features

- 🗂 **20 concept clusters** — related concepts grouped across all 3 layers
- 🔍 **Search** — find any concept by name or description instantly
- 🎛 **Filter by layer** — focus on L1, L2, or L3 at a time
- 📋 **Map & List views** — grid card view or compact flat list
- 💬 **Concept detail panel** — click any concept for its definition and context
- 📱 **Responsive** — works on desktop and mobile

## Running Locally

**Requirements:** Node.js 18+ installed

```bash
# Clone the repo
git clone https://github.com/Peter-AIExplorer/python-mindmap.git
cd python-mindmap

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open `http://localhost:5173` in your browser.

## About

Built as a personal Python learning reference. The concept list was assembled by mapping three layers of knowledge:
- What applies to **all programming** (paradigms, algorithms, memory)
- What is specific to **Python as a platform** (environments, packaging, tooling)
- What is specific to **Python the language** (syntax, built-ins, standard library)

Feel free to fork it, adapt it for your own notes, or suggest concepts that are missing.
